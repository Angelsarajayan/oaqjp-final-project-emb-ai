# Emotion Detection Application

## 📌 Project Overview
This project implements an **Emotion Detection Web Application** that analyzes user input text and detects emotions using Natural Language Processing (NLP).

The system identifies the following emotions:
- Anger
- Disgust
- Fear
- Joy
- Sadness

The application is deployed using a Flask web server and provides emotion analysis through a web interface.

---

## Features
- Emotion detection from user text input
- Identification of dominant emotion
- Web-based interface using Flask
- Error handling for blank inputs
- Unit testing implementation
- Static code analysis using PyLint (10/10 score)

---

## Technologies Used
- Python 3
- Flask
- REST API
- JSON Processing
- PyLint
- Git & GitHub

---
## How to Run the Project

### Step 1: Install Required Libraries
pip install flask, pylint
### Step 2: Run the Application
python3 server.py
### Step 3: Open in Browser
http://localhost:5000
